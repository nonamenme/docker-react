### setup
```sh
$ git clone git@github.com:nonamenme/docker-react.git
$ cd docker-react
```

### command
・Image build
```sh
$ docker-compose build
```

・Container start up
```sh
$ docker-compose up -d
```

・Container stop
```sh
$ docker-compose stop
```

・Container remove
```sh
$ docker-compose down
```

### URL
http://localhost:3000
